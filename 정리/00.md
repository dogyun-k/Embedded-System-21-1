# Ubuntu Terminal Env 사용

- vim

        :wq : 저장 후 나가기
        :sp : 화면 분할

        dd  : 한줄 삭제 (Ctrl + x)
        3dd : 세줄 삭제
        yy  : 한줄 복사 (Ctrl + c)
        3yy : 세줄 복사
        p   : 붙여넣기 (Ctrl + v)
        u   : 되돌리기 (Ctrl + z)
        o   : 다음 줄에 커서 이동
        r   : 그 칸 대체입력
        dw  : 한개 단어 지우기

        /[검색어]   : 검색하기
                n   : 다음 검색어  

        Shift + j   : 모아주기

        $   : 마지막줄로

        :1,$s/[바꾸고 싶은 단어]/[바꿀 단어]/g  : 모두바꾸기
        /c  : 바꾸는거 물어봄
